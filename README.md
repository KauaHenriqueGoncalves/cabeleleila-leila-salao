# Leila Salão

Projeto desenvolvido para gerenciamento de clientes, serviços e agendamentos de um salão de beleza da Leila.

O repositório está organizado em duas áreas principais:

### [Apresentation](./apresentation)

Contém os materiais de apresentação do projeto.

Nesta pasta podem ser encontrados:

- Capturas de tela da aplicação
- Vídeos demonstrativos
- Imagens das funcionalidades implementadas

### [source-code](./source-code)

Contém todo o código-fonte da solução.

Nesta pasta estão presentes:

- Backend (API)
- Frontend (Angular)
- Arquivos Docker
- Configurações de ambiente
- Documentação técnica
